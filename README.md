# Projet-clustering-nba
Identification d'une équipe type gagnante par clustering

Objectif principal :

Identifier une équipe type NBA performante en analysant les profils statistiques des joueurs par poste à l’aide de méthodes de clustering non supervisé, à partir des saisons 2010 à 2026.

Objectifs secondaires :

Comprendre les différents archétypes de joueurs par poste

Étudier l’évolution des profils NBA sur la période

Relier certains profils à la performance collective

Période analysée

Saisons : 2010 à 2025

Justification :

Début de l’ère analytique moderne (3pts, pace, spacing)

Suffisamment long pour observer des tendances stables

Population étudiée

Joueurs NBA ayant :

Un poste identifié (PG, SG, SF, PF, C)

Un temps de jeu significatif

Définition des postes

Postes standards NBA :

PG — Point Guard

SG — Shooting Guard

SF — Small Forward

PF — Power Forward

C — Center

Hypothèses de travail:
Hypothèse 1 — Existence d’archétypes par poste

Pour chaque poste, il existe plusieurs profils statistiques distincts (scoreur, défenseur, playmaker, etc.) détectables par clustering.

Hypothèse 2 — Tous les profils ne contribuent pas autant à la victoire

Certains clusters sont surreprésentés dans les équipes performantes (playoffs, contenders).

Hypothèse 3 — Une équipe gagnante est une combinaison optimale de profils

Une équipe performante peut être modélisée comme une combinaison spécifique de clusters, un par poste.
